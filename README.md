# Cyber Incident Manager

## Overview
This is a database-driven web application used to track cybersecurity incidents.  
Users can create, view, and update incidents, along with associated evidence and tasks.

---

## Features
- Create new incidents
- View all incidents
- Update incident details
- Add evidence and tasks
- Run SQL queries (join, aggregate, delete)

---

## Technologies
- Python (Flask)
- SQLite
- HTML / CSS
